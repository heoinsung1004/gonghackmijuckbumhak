1. 합성함수 $f \circ g$의 도출 및 정의역 분석

[1단계 (What): 계산 과정 및 근거 설명]
계산 과정:함수 $f(g(x))$는 $f(\frac{1}{x})$이므로, $f(g(x)) = \sqrt{\frac{1}{x} + 1} = \sqrt{\frac{1+x}{x}}$이다. 정의역을 구하기 위해 두 가지 조건을 고려한다.g(x)가 정의되어야 하므로 분모 $x \neq 0$이다.무리함수 $f$의 내부 식(피제곱근수)이 0 이상이어야 하므로 $\frac{1+x}{x} \ge 0$이다. 부등식 $\frac{1+x}{x} \ge 0$을 풀면 $x \le -1$ 또는 $x > 0$이다. (분모가 0인 지점 제외)따라서 $f \circ g$의 정의역은 (${x \mid x \le -1 \text{ 또는 } x > 0}$)이다.

근거 설명: 

합성함수의 정의: $(f \circ g)(x) = f(g(x))$

무리함수의 실수 조건: 근호 안의 식은 항상 0 이상이어야 함. 

유리함수의 존재 조건: 분모는 0이 될 수 없음.

[2단계 (So What): 결과 해석]결과 해석: 이 결과는 합성함수의 정의역이 단순히 최종 형태의 식뿐만 아니라, 내부 함수($g$)의 정의역에 의해 먼저 제한됨을 보여준다. 기하학적으로 볼 때, $x=0$ 부근에서 함수가 정의되지 않는 '불연속 구간'이 발생하며, 이는 특정 입력값이 시스템의 첫 번째 필터($g$)를 통과하지 못할 경우 전체 시스템($f \circ g$)이 작동 불능 상태가 됨을 공학적으로 시사한다.



2. 함수 $F(x) = \sqrt{|x^3 - 1| + 3}$를 두 함수의 합성함수 형태로 표현하여라.

[1단계 (What): 계산 과정 및 근거 설명]

계산 과정:주어진 함수 $F(x) = \sqrt{|x^3 - 1| + 3}$을 내부 연산과 외부 연산으로 층위(Layer)를 나눈다. 가장 안쪽의 연산 뭉치를 내부 함수 $g(x)$로 설정한다: $g(x) = |x^3 - 1| + 3$

$g(x)$를 입력값으로 받아 최종 연산을 수행하는 외부 함수 $f(x)$를 설정한다: $f(x) = \sqrt{x}$

두 함수를 합성하면 $(f \circ g)(x) = f(g(x)) = f(|x^3 - 1| + 3) = \sqrt{|x^3 - 1| + 3}$이 되어 $F(x)$와 일치함을 확인한다. (참고: $g(x) = x^3 - 1$, $f(x) = \sqrt{|x| + 3}$과 같이 다른 방식으로도 분해가 가능하다.)


근거 설명:합성함수의 정의: 두 함수 $f$와 $g$에 대하여 $(f \circ g)(x) = f(g(x))$로 정의된다.
함수의 계층 구조: 복잡한 형태의 기본 함수(Elementary function)들의 조합을 연산 순서에 따라 분리하여 표현할 수 있다.


[2단계 (So What): 결과 해석]결과 해석: 이 문제는 복잡한 형태의 함수를 더 단순하고 분석하기 쉬운 기초적인 두 함수의 결합체로 구조화하는 과정을 보여준다. 수학적 관점에서 이러한 분해는 전체 함수의 치역(Range)을 결정할 때 내부 함수의 치역이 외부 함수의 정의역으로 전이되는 연쇄적 흐름을 명확하게 파악할 수 있게 하며, 이는 함수의 연속성이나 미분 가능성을 국소적으로 판단하는 데 필수적인 기초 분석 단계가 된다.


3. (연습문제 1.2-8) 자연로그 역함수

[문제]함수 $f(x) = e^{x-1}$의 역함수를 구하여라.

[1단계 (What): 계산 과정 및 근거 설명]

계산 과정:주어진 함수를 $y = e^{x-1}$로 둔다.식을 $x$에 관하여 풀기 위해 양변에 자연로그($\ln$)를 취하면 $\ln(y) = \ln(e^{x-1})$이 된다.로그의 성질에 의해 $\ln(y) = x - 1$이 되고, $x$에 대하여 정리하면 $x = \ln(y) + 1$이다.역함수를 구하기 위해 $x$와 $y$의 자리를 바꾸면 $y = \ln(x) + 1$이다.원래 지수함수의 치역이 항상 $y > 0$이므로, 구하는 역함수의 정의역은 $x > 0$이 된다. 따라서 역함수는 $f^{-1}(x) = \ln(x) + 1 \ (x > 0)$이다.

근거 설명:역함수의 존재 조건: 지수함수는 실수 전체 구간에서 일대일 대응이므로 역함수가 존재한다.지수와 로그의 관계: 자연지수함수와 자연로그함수는 서로 역연산 관계이므로 $\ln(e^x) = x$가 성립한다.

[2단계 (So What): 결과 해석]결과 해석: 이 풀이는 원래 함수와 역함수가 직선 $y=x$에 대하여 기하학적으로 대칭을 이룬다는 사실을 수식으로 증명한다. 원래 함수 $f(x) = e^{x-1}$은 기본 지수함수인 $y=e^x$를 $x$축 방향으로 1만큼 평행이동한 형태이다. 이를 통해 도출된 역함수 $f^{-1}(x) = \ln(x) + 1$은 기본 로그함수 $y=\ln(x)$를 $y$축 방향으로 1만큼 평행이동한 형태가 된다. 결론적으로 원래 함수에서의 $x$축 방향 이동이, 역함수에서는 $y$축 방향 이동으로 그대로 전이됨을 확인할 수 있다.

4. (연습문제 1.3-1) 쌍곡함수

[문제] sinh(ln 2) + cosh(ln3)의 값을 구하여라.

[1단계 (What): 계산 과정 및 근거 설명]계산 과정:쌍곡사인과 쌍곡코사인 함수의 정의를 이용하여 주어진 식을 지수 형태로 변환한다. $\sinh(\ln 2) = \frac{e^{\ln 2} - e^{-\ln 2}}{2}$ 이고, $\cosh(\ln 3) = \frac{e^{\ln 3} + e^{-\ln 3}}{2}$ 이다. 로그의 성질에 따라 $e^{\ln x} = x$ 이고 $e^{-\ln x} = x^{-1} = \frac{1}{x}$ 이므로 이를 대입한다. $\sinh(\ln 2)$의 값을 계산하면 $\frac{2 - \frac{1}{2}}{2} = \frac{\frac{3}{2}}{2} = \frac{3}{4}$ 이다. $\cosh(\ln 3)$의 값을 계산하면 $\frac{3 + \frac{1}{3}}{2} = \frac{\frac{10}{3}}{2} = \frac{5}{3}$ 이다. 두 값을 더하면 최종 결과는 $\frac{3}{4} + \frac{5}{3} = \frac{9}{12} + \frac{20}{12} = \frac{29}{12}$ 이다.

근거 설명:쌍곡함수의 정의: $\sinh(x) = \frac{e^x - e^{-x}}{2}$ 이며, $\cosh(x) = \frac{e^x + e^{-x}}{2}$ 이다. 지수와 로그의 역연산 성질: 밑이 무리수 $e$인 지수함수와 자연로그는 서로 역함수 관계이므로, $e^{\ln a} = a$ 및 $e^{-\ln a} = \frac{1}{a}$ 이 성립한다.

[2단계 (So What): 결과 해석]결과 해석:이 계산 과정은 초월함수인 쌍곡함수에 무리수인 자연로그 값을 대입했을 때, 최종 결과가 단순한 유리수 형태로 도출됨을 보여준다. 이는 쌍곡함수가 본질적으로 자연지수함수의 결합으로 정의되어 있기 때문에 발생하는 현상이다. 해석학적 관점에서, 복잡해 보이는 쌍곡선 위의 특정 좌표들이 자연로그라는 매개변수를 거칠 때 대수적으로 다루기 쉬운 유리수 값으로 환원될 수 있음을 증명하는 사례이다.

5. (연습문제 1.4-6) 역삼각함수

[문제] $$\sin \left[ \cos^{-1} \left(\frac{4}{5}\right) + \cos^{-1} \left(\frac{3}{5}\right) \right]$$ 의 값을 구하여라.

[1단계 (What): 계산 과정 및 근거 설명]계산 과정:계산의 편의를 위해 식 내부의 역삼각함수를 각각 $\alpha$ 와 $\beta$ 로 치환한다. 즉, $\cos^{-1} (4/5) = \alpha$, $\cos^{-1} (3/5) = \beta$ 로 둔다.역삼각함수의 정의에 의해 $\cos \alpha = 4/5$ 이고 $\cos \beta = 3/5$ 가 된다. 주어진 값이 모두 양수이므로 $\alpha$ 와 $\beta$ 는 제1사분면의 각이다. 삼각함수의 제곱 관계를 이용하여 $\sin \alpha$ 와 $\sin \beta$ 의 값을 구한다. 제1사분면이므로 사인 값은 양수이다.

$\sin \alpha = \sqrt{1 - \cos^2 \alpha} = \sqrt{1 - (4/5)^2} = 3/5$ 이다. 

$\sin \beta = \sqrt{1 - \cos^2 \beta} = \sqrt{1 - (3/5)^2} = 4/5$ 이다. 

원래 구하고자 했던 식은 $\sin(\alpha + \beta)$ 가 되므로, 삼각함수의 덧셈정리를 적용한다.  $\sin(\alpha + \beta) = \sin \alpha \cos \beta + \cos \alpha \sin \beta$ 공식에 값을 대입한다. $(3/5 \cdot 3/5) + (4/5 \cdot 4/5) = 9/25 + 16/25 = 25/25 = 1$ 이 된다. 

근거 설명:역삼각함수의 정의: $y = \cos^{-1}(x)$ 이면 $x = \cos y$ 가 성립한다. 삼각함수의 제곱 관계: 모든 각에 대하여 $\sin^2 \theta + \cos^2 \theta = 1$ 이 성립한다. 삼각함수의 덧셈정리: 사인 함수의 합 공식인 $\sin(\alpha + \beta) = \sin \alpha \cos \beta + \cos \alpha \sin \beta$ 를 적용한다.


[2단계 (So What): 결과 해석]결과 해석: 이 풀이는 역삼각함수로 표현된 각들을 덧셈정리를 통해 유리수 계산으로 변환하는 과정을 보여준다. 계산 과정에서 $\cos \alpha$ 와 $\sin \alpha$ 의 값이 서로 교차되어 나타나는 점에 주목해야 한다.기하학적으로 이는 세 변의 길이가 3, 4, 5인 직각삼각형에서 두 예각의 합을 구하는 것과 같다. 따라서 두 각의 합인 $\alpha + \beta$ 는 정확히 $90^\circ(\pi/2)$ 가 되며, 최종적으로 $\sin(90^\circ) = 1$ 이라는 결과가 도출된다. 이는 수치적 계산이 기하학적 성질과 완벽히 일치함을 증명한다.


6. (예제 1.4-15) 역쌍곡사인함수[문제] $$\sinh^{-1} x = \ln(x + \sqrt{x^2 + 1})$$임을 증명하여라.

[1단계 (What): 계산 과정 및 근거 설명]계산 과정:증명하고자 하는 역쌍곡사인함수를 $y$로 치환한다. 즉, $y = \sinh^{-1} x$ 로 둔다. 역함수의 성질에 따라 $x = \sinh y$ 가 성립한다.쌍곡사인함수의 정의를 대입하여 식을 전개한다. $$x = \frac{e^y - e^{-y}}{2}$$식의 양변에 $2e^y$ 를 곱하여 $e^y$ 에 대한 이차방정식 형태로 정리한다. $$2x e^y = (e^y)^2 - 1$$$$(e^y)^2 - 2x e^y - 1 = 0$$ 근의 공식을 사용하여 $e^y$의 값을 구한다. $$e^y = \frac{2x \pm \sqrt{(-2x)^2 - 4(1)(-1)}}{2} = \frac{2x \pm 2\sqrt{x^2 + 1}}{2} = x \pm \sqrt{x^2 + 1}$$ 지수함수의 성질에 의해 $e^y$는 항상 양수이어야 한다. 모든 실수 $x$에 대하여 $\sqrt{x^2 + 1} > |x|$ 이므로, $x - \sqrt{x^2 + 1}$ 은 항상 음수이다. 따라서 양수인 근만 선택한다. $$e^y = x + \sqrt{x^2 + 1}$$양변에 자연로그를 취하여 $y$에 대하여 식을 정리한다. $$y = \ln(x + \sqrt{x^2 + 1})$$처음에 설정한 $y = \sinh^{-1} x$ 를 대입하면 주어진 등식이 성립함이 증명된다.

근거 설명:역함수의 정의: $y = f^{-1}(x)$ 이면 $x = f(y)$ 가 성립한다. 쌍곡사인함수의 정의: $\sinh y = \frac{e^y - e^{-y}}{2}$ 로 정의된다. 지수함수의 성질: 모든 실수 $y$에 대하여 지수함수의 값 $e^y > 0$ 이 항상 성립한다.

[2단계 (So What): 결과 해석]
이 증명 결과는 도출된 로그 형태의 수식이 실수 전체 구간에서 모순 없이 정의됨을 확인해 준다. 로그가 성립하기 위해서는 진수인 $x + \sqrt{x^2 + 1}$ 이 항상 0보다 커야 한다. 여기서 무리식 $\sqrt{x^2 + 1}$ 은 항상 $|x|$ 보다 크기 때문에, $x$ 가 음수일 때조차도 두 값의 합은 반드시 양수가 되어 진수 조건을 완벽하게 만족시킨다. 이는 원래 함수인 쌍곡사인함수의 치역이 실수 전체이므로, 그 역함수의 정의역 또한 실수 전체가 되어야 한다는 대수적 성질과 도출된 공식이 정확히 일치함을 보여준다.


7. (연습문제 3.1-1) 연속이며 미분 가능한 함수

[문제]함수의 연속성과 미분 가능성을 조사하여라.

[1단계 (What): 계산 과정 및 근거 설명]계산 과정 (연속성 조사): $x = 0$ 에서의 함숫값은 주어진 정의에 따라 $f(0) = 0$ 이다. $x \to 0$ 일 때의 극한값을 구하기 위해 부등식을 세운다. 사인 함수의 성질에 의해 항상 다음이 성립한다. $$-1 \le \sin \left(\frac{1}{x}\right) \le 1$$양변에 항상 0 이상인 $x^2$ 을 곱하면 부등호의 방향이 유지된다. $$-x^2 \le x^2 \sin \left(\frac{1}{x}\right) \le x^2$$극한 $\lim_{x \to 0} (-x^2) = 0$ 이고 $\lim_{x \to 0} (x^2) = 0$ 이므로 조임 정리에 의해 극한값은 0으로 수렴한다. $$\lim_{x \to 0} x^2 \sin \left(\frac{1}{x}\right) = 0$$원점에서의 극한값(0)과 함숫값(0)이 일치하므로 $f(x)$ 는 $x=0$ 에서 연속이다. ($x \neq 0$ 인 구간은 연속함수의 곱이므로 당연히 연속이다.) 계산 과정 (미분 가능성 조사):미분계수의 정의를 이용하여 $x = 0$ 에서의 도함수 값 $f'(0)$ 을 구하는 극한식을 세운다. $$f'(0) = \lim_{h \to 0} \frac{f(0+h) - f(0)}{h} = \lim_{h \to 0} \frac{h^2 \sin(1/h) - 0}{h}$$분모와 분자의 $h$를 약분하여 식을 정리한다. $$f'(0) = \lim_{h \to 0} h \sin \left(\frac{1}{h}\right) $$연속성 증명과 동일하게 조임 정리를 적용한다. $-|h| \le h \sin(1/h) \le |h|$ 이고 양끝 항이 0으로 수렴하므로 이 극한값 역시 0이 된다. $$\lim_{h \to 0} h \sin \left(\frac{1}{h}\right) = 0$$미분계수 $f'(0) = 0$ 의 값이 존재하므로 이 함수는 $x=0$ 에서 미분 가능하다. 결론적으로 $f(x)$는 모든 실수 구간에서 연속이며 미분 가능하다.

근거 설명:연속의 정의: 극한값 $\lim_{x \to a} f(x)$ 가 존재하고 그 값이 함숫값 $f(a)$ 와 같으면 함수는 $x=a$ 에서 연속이다.미분 가능성의 정의: 평균변화율의 극한인 미분계수 $\lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$ 가 특정한 실수 값으로 존재할 때 미분 가능하다고 한다.조임 정리 (압착 정리): $g(x) \le f(x) \le h(x)$ 이고 양끝 함수의 극한값이 같으면, 가운데 함수의 극한값도 동일한 값으로 수렴한다.

[2단계 (So What): 결과 해석]결과 해석:이 문제는 원점 근처에서 무한히 진동하여 극한을 정의할 수 없는 불안정한 함수 $\sin(1/x)$ 가, $x^2$ 이라는 강력한 다항함수 형태의 경계선(포락선) 안에 갇히면서 어떻게 수학적으로 안정화되는지를 해석학적으로 증명한다. $x^2$ 은 원점에서 접선의 기울기가 0으로 매우 완만하게 좁아지는 포물선이다. 따라서 이 두 함수가 곱해지면 원점으로 갈수록 진동 폭이 급격하게 압착된다. 그 결과, 끊어짐 없이 부드럽게 이어지는 '연속' 상태가 될 뿐만 아니라, 원점에서의 순간적인 기울기마저 0으로 평탄해져 뾰족한 점(첨점) 없이 '미분 가능'한 상태로 매끄럽게 연결된다는 것을 대수적 수식을 통해 명확히 확인할 수 있다.



8. (연습문제 3.1-11) 절댓값 함수의 도함수

[문제]도함수의 정의와 $|x| = \sqrt{x^2}$ 을 이용하여 다음을 증명하여라. $$\frac{d}{dx}|x| = \frac{|x|}{x} \quad (x \neq 0)$$

[1단계 (What): 계산 과정 및 근거 설명]계산 과정: 주어진 힌트에 따라 절댓값 함수를 지수 형태로 변환한다. $$f(x) = |x| = \sqrt{x^2} = (x^2)^{\frac{1}{2}}$$ 합성함수의 미분법을 적용하여 도함수를 구한다. 겉함수는 지수가 $1/2$ 인 함수이고, 속함수는 $x^2$ 이다. $$\frac{d}{dx} (x^2)^{\frac{1}{2}} = \frac{1}{2}(x^2)^{-\frac{1}{2}} \cdot \frac{d}{dx}(x^2)$$ 속함수의 미분 결과인 $2x$ 를 곱하여 식을 정리한다. $$\frac{1}{2} \frac{1}{\sqrt{x^2}} \cdot 2x = \frac{x}{\sqrt{x^2}}$$원래의 힌트인

$\sqrt{x^2} = |x|$ 를 분모에 다시 대입한다.

$$\frac{x}{|x|}$$ $x \neq 0$ 이므로, 분모와 분자에 동일하게 $|x|$ 를 곱하여 식을 변형한다. $x^2 = |x|^2$ 이 성립하므로 최종 결과가 도출된다. $$\frac{x \cdot |x|}{|x| \cdot |x|} = \frac{x |x|}{x^2} = \frac{|x|}{x}$$ 

근거 설명:절댓값의 성질: 임의의 실수 $x$ 에 대하여 $|x| = \sqrt{x^2}$ 이 성립한다. 합성함수의 미분법: 두 함수가 합성된 경우 겉함수의 미분에 속함수의 미분을 곱하여 계산한다.

[2단계 (So What): 결과 해석]결과 해석:이 증명은 꺾인 선 형태인 절댓값 함수를 매끄러운 다항함수의 합성 형태로 치환함으로써 대수적인 미분 계산을 가능하게 한다.도출된 최종 결과식 $\frac{|x|}{x}$ 는 $x > 0$ 일 때 $1$, $x < 0$ 일 때 $-1$ 의 값을 가진다. 이는 원점($x=0$)을 기준으로 좌극한 기울기($-1$)와 우극한 기울기($1$)가 급격하게 교차함을 수식으로 명확히 보여주는 것이다.
